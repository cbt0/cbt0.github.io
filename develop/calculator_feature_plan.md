# 계산기 기능 계획서

## 목표
문제 풀이 중 빠르게 계산할 수 있도록 계산기 기능을 추가합니다. 계산기는 괄호 계산, 제곱/삼제곱, 루트, ln, log, 지수 연산 등을 지원하여 에너지, 가스, 공조 문제 풀이에 필요한 수식 계산을 도와줍니다.

## 기능 요구사항
- 문제 풀이 화면 상단에 `계산기` 버튼 추가
- 클릭 시 계산기 모달이 열림
- 계산기 입력은 화면에 표시되며, `=` 버튼으로 계산 결과 표시
- `C` 버튼으로 초기화
- 지원 연산
  - 기본 사칙연산: `+`, `-`, `×`, `÷`
  - 괄호: `(`, `)`
  - 소수점: `.`
  - 제곱: `x²`, `x³`, `^` (지수)
  - 루트: `√` => `sqrt()`
  - 자연로그: `ln()`
  - 상용로그: `log()`
- `=` 계산 시 JavaScript `Function`을 사용해 안전하게 평가

## UI 구성
- 버튼: `C (초기화)`, `(`, `)`, `÷`
- 버튼: `7`, `8`, `9`, `×`
- 버튼: `4`, `5`, `6`, `x²`
- 버튼: `1`, `2`, `3`, `x³`
- 버튼: `0`, `.`, `√`, `+`
- 버튼: `ln`, `log`, `^`, `=`

## 구현 계획
1. `index.html`에 계산기 모달 구조 추가
2. `css/style.css`에 계산기 스타일 정의
3. `js/app.js`에 계산기 DOM 요소 참조 추가
4. `registerEventListeners()`에 계산기 버튼 이벤트 등록
5. `handleCalculatorInput(value)` 함수 작성
6. `evaluateCalculatorExpression(expr)` 함수 작성
7. 계산기 모달 닫기 및 백그라운드 클릭 시 닫기 구현
8. 동작 검증: 괄호 계산, 제곱/삼제곱, 루트, ln, log 계산 확인

## 테스트 시나리오
- `2+3*4` -> `14`
- `(1+2)*3` -> `9`
- `sqrt(16)` -> `4`
- `ln(1)` -> `0`
- `log(100)` -> `2`
- `2^3` -> `8`
- `5x²` 입력 후 `=` -> `25`
- `C` 클릭 시 `0`으로 초기화

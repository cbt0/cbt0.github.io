# CBT 문제풀이 사이트 제작 계획서 (Plan)

이 계획서는 **최강 CBT** 스타일의 자격증 기출문제 학습 사이트인 `cbt0.github.io`를 구축하고 고도화하기 위한 상세 개발 계획서입니다. 현재 가스기능사, 에너지관리기능장, 에너지관리산업기사의 총 60회차(4,040문제) 실제 데이터를 구축 완료하고 사이트의 주요 편의 기능 및 다중 사용자 세션 관리 기능을 적용하여 배포 중에 있습니다.

---

## 1. 프로젝트 개요
- **목표**: 에너지/가스/공조 분야 기출문제를 풀 수 있는 반응형 최강 CBT 웹 어플리케이션 구축
- **핵심 기술**: HTML5, Vanilla CSS (Premium Dark Mode & Glassmorphism), JavaScript (ES6+, SPA 구조)
- **배포 환경**: GitHub Pages (`https://cbt0.github.io`)
- **버전 규칙**: Push 배포 시마다 버전을 `0.1`씩 상향 조정 (현재 배포 버전: **CBT V1.2**)

---

## 2. 웹사이트 메뉴 및 레이아웃 구성

사이트는 상단 헤더바를 통해 **[홈(소개/과목선택)]**, **[문제(CBT 풀이)]**, **[채점(개인 통계/OMR판)]**, **[설정(테마)]**의 4가지 주요 탭을 동적으로 오가는 Single Page Application(SPA) 구조를 갖춥니다.

1. **홈 (Home) / 소개 및 과목선택**
   - **로그인 전**: 사이트 타이틀 및 가이드, **사용자 로그인 패널**(ID 및 비밀번호 입력창) 노출
   - **로그인 후**: 환영 메시지 및 로그아웃 버튼 노출, **[학습 과목 선택]** 그리드 활성화 및 스크롤 포커스 이동
2. **가스기능사 (실제 기출)**
   - 2017년 ~ 2025년 최신 기출 복원문제 완벽 탑재 (18회차 / 1,080문제, 표 서식 완벽 복구)
3. **에너지관리산업기사 (실제 기출)**
   - 2012년 ~ 2020년 실제 기출문제 완벽 탑재 (22회차 / 1,760문제, 문단 서식 완벽 복구)
4. **에너지관리기능장 (실제 기출)**
   - 실전모의고사 기출문제 완벽 탑재 (20회차 / 1,200문제, 표 서식 완벽 복구)
5. **에너지기능사 (준비 중 / 샘플 기출)**
   - 기초 샘플 문제 제공 및 추가 데이터 준비 중 안내
6. **공조냉동기계기능사 (준비 중 / 샘플 기출)**
   - 기초 샘플 문제 제공 및 추가 데이터 준비 중 안내

## 3. 핵심 기능 요구사항 및 아키텍처

- **사용자 인증 (Login System)**:
  - ID와 비밀번호를 사용하여 로컬 환경에서 인증 처리 (초기 비밀번호: `"dongbu"`).
  - 인증 성공 시에만 과목 선택 카드가 활성화되며, 로그인 정보는 로컬 스토리지에 유지되어 새로고침 후에도 자동 세션 연동.
- **개인 학습 통계 대시보드 ("채점" 탭)**:
  - 기존 홈 화면에 고정되어 있던 **3대 통계(푼 문제 수, 평균 정답률, 합격한 회차)** 지표를 "채점" 탭으로 이전.
  - 해당 통계 지표는 로그인한 **사용자 ID별로 분리되어 독립적으로 기록 및 적재**되도록 아키텍처 설계.
  - 시험 진행 중에는 OMR 마킹 현황판이 노출되며, 시험이 활성화되지 않은 상태에서 "채점" 탭 클릭 시에는 개인 학습 통계 대시보드가 표시됨.
- **마지막 푼 문제 "이어 풀기" (Resume Session)**:
  - 사용자가 문제를 풀거나 탐색할 때마다 해당 사용자 ID의 마지막 학습 로그(과목, 회차, 문제 인덱스, 타임스탬프)를 실시간 자동 캐싱.
  - "채점" 탭의 개인 대시보드에 마지막 풀던 문제 정보와 **[이어 풀기]** 버튼을 노출하여, 클릭 즉시 풀던 문제 번호와 시험 시간(타이머)이 그대로 재개되도록 조율.
- **상세 활동 로그 수집 (User Logs)**:
  - 로그인 성공, 로그아웃, 시험 시작, 각 회차별 시험 제출 결과(점수 및 합격여부) 등의 이력을 실시간 타임스탬프와 함께 수집하여 개인 대시보드 하단에 스크롤 가능한 목록으로 실시간 출력.
- **CBT 풀이 및 실시간 채점 엔진**:
  - 문항 선택 시 실시간 채점 피드백(초록/빨강 시각 효과) 제공 및 해당 문제의 해설 박스 자동 확장.
  - 문제 지문 및 해설에 포함된 표(Table) 구조가 깨지지 않도록 `innerHTML`과 `white-space: normal;` 스타일 설정을 연동하여 완벽한 표 레이아웃 구현.
  - 제출 시 SVG stroke-dashoffset 애니메이션을 차용한 합격 여부 원형 게이지 모달 리포트 제공.

## 4. UI/UX 디자인 시스템 (Premium Dark UI)
- **색상 테마 (Sleek Dark Mode & Violet Gold Accent)**:
  - 메인 배경: `#0f172a`, `#1e293b` (Slate Dark Gray)
  - 액센트: `#6366f1` (Indigo Violet), `#0984e3` (Blue)
  - 상태값: `#10b981` (Emerald Green, 정답/합격), `#f43f5e` (Rose Red, 오답/불합격)
- **글라스모피즘 스타일링**:
  - 투명한 유리 패널 질감(`backdrop-filter: blur(10px)`)과 얇은 테두리(`border: 1px solid rgba(255, 255, 255, 0.08)`)를 사용하여 현대적이고 고급스러운 느낌 전달.
- **모바일 웹 최적화**:
  - 모바일 환경(768px 이하)에서 여백을 최소화하고 문제 폰트 크기(지문 18px, 보기 16px)를 극대화하여 뛰어난 가독성 보장.
  - OMR 판 and 문제지를 모바일 해상도에서는 탭(Tab) 기반으로 1단 정렬 노출되도록 미디어 쿼리 최적화.

## 5. 데이터 파싱 결과 요약
- **파싱 프로그램**: `parse_docx_to_json.py` (xml/zip 파싱 및 nested-table HTML화) & `parse_energy_sanup_to_json.py` (State Machine 기반 스타일 런 파싱)
- **가스기능사**: 18개 회차 × 60문제 = **총 1,080문제** (HTML 표 복구 완료)
- **에너지관리산업기사**: 22개 회차 × 80문제 = **총 1,760문제** (글꼴 서식 및 요약 표 복구 완료)
- **에너지관리기능장**: 20개 회차 × 60문제 = **총 1,200문제** (HTML 표 복구 완료)
- **산출 위치**: `data/gas/`, `data/energy_ginungjang/`, `data/energy_sanupgisa/` 디렉토리 내 JSON 파일로 저장

## 6. 물리적 폴더 및 파일 구조

현재의 구조는 다중 과목 추가 탑재 및 자원 관리의 확장성을 고려하여 다음과 같이 엄격하게 정립되었습니다.

```text
cbt0.github.io/
├── index.html              # 메인 엔트리 (SPA 라우터 및 HTML5 구조)
├── plan.md                 # [현재 파일] 프로젝트 개발 계획 및 설계서
├── README.md               # 프로젝트 간략 소개
├── css/
│   └── style.css           # UI/UX, 테마 및 모바일 반응형 전용 스타일시트
├── js/
│   └── app.js              # 비동기 DB 로드, 로그인, 타이머, OMR 및 퀴즈 상태 제어 코어
├── data/
│   ├── gas/
│   │   └── gas_questions.json          # 가스기능사 기출 데이터베이스 (18회차)
│   ├── energy_ginungjang/
│   │   └── energy_ginungjang_questions.json # 에너지관리기능장 기출 데이터베이스 (20회차)
│   ├── energy_sanupgisa/
│   │   └── energy_sanupgisa_questions.json  # 에너지관리산업기사 기출 데이터베이스 (22회차)
│   └── source/
│       ├── gas_ginungsa_CBT_2017_2025_Final.docx # 원본 가스 docx 파일
│       ├── energy_ginungjang_01_20.docx          # 원본 기능장 docx 파일
│       └── enegy_sanup_gisa_v3.docx              # 원본 산업기사 docx 파일
└── develop/
    ├── database_integration_plan.md # 글로벌 DB 및 순위 연동 기획서
    ├── develop.md          # 개발 이력, 아키텍처 및 상세 디버깅 가이드
    └── history.md          # 이전 CLI 작업 내역 히스토리 로그
```
